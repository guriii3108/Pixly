## üì¶ Installation & Setup

1.  **Clone the repository**

    ```bash
    git clone https://github.com/yourusername/pixly.git
    cd pixly
    ```

2.  **Install dependencies**

    ```bash
    npm install
    ```

3.  **Environment Setup**
    Create a `.env` file in the root directory and add your API keys:

    ```env
    VITE_UNSPLASH_KEY=your_unsplash_access_key
    VITE_PEXELS_KEY=your_pexels_api_key
    VITE_TENOR_KEY=your_tenor_api_key
    ```

4.  **Run the development server**

    ```bash
    npm run dev
    ```

5.  **Build for production**
    ```bash
    npm run build
    ```

## üé® Design Philosophy

Pixly follows a **"Sophisticated Minimalist"** design philosophy.

-   **Typography**: Clean, sans-serif fonts with generous tracking for a cinematic feel.
-   **Color Palette**: Deep Zinc (`zinc-950`) backgrounds with white text and subtle gradient accents.
-   **Motion**: Deliberate, smooth transitions (`ease-out`) rather than jumpy animations.
